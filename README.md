# CS-GradeCalculator

Aplicación de consola para registrar evaluaciones, asistencia, acuerdos docentes y calcular notas finales.

## Requisitos previos
- Node.js 18+

## Instalación
```bash
npm install
```

## Ejecutar la aplicación (CLI)
```bash
npm start
```
Sigue el menú interactivo para registrar datos y calcular la nota final.

## Ejecutar tests con cobertura
```bash
npm test
```
Genera `coverage/lcov.info` para Sonar y muestra el resumen en consola.
